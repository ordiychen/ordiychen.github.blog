About Apache Ambari

Apache Ambari 是Hortonworks贡献给Apache基金会的Hadoop平台管理软件，2013年11月20日成为Apache顶级项目。它具备Hadoop组件的安装、管理、运维等基本功能，提供Web UI进行可视化的集群管理，简化了大数据平台的安装、使用难度。

Apache Ambari 依赖于HDP的VDF
VDF文件
Ambari Server使用版本定义文件（VDF）来了解发行版中包含哪些产品和组件版本。VDF文件包含各种操作系统及其回购URL。VDF还提供了该产品版本附带的各个组件名称和版本信息。
示例： HDP 2.6.2（VDF）：http : //public-repo-1.hortonworks.com/HDP/centos6/2.x/updates/2.6.5.0/HDP-2.6.5.0-292.xml
